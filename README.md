# Micro-CT Vocabulary Matching Game

A simple matching game made with HTML, CSS, and JavaScript.

## What it does
- Shows Micro-CT vocabulary on one side
- Shows definitions on the other side
- Lets the user match terms to definitions
- Gives a final score
- Lists the questions that were wrong

## Files
- `index.html` - page structure
- `styles.css` - styling
- `script.js` - game logic

## How to run locally
1. Download the folder
2. Open `index.html` in your browser

## How to put it on GitHub
1. Create a new GitHub repository
2. Upload `index.html`, `styles.css`, `script.js`, and `README.md`
3. Commit the files
4. Go to **Settings** -> **Pages**
5. Under **Build and deployment**, choose **Deploy from a branch**
6. Select the `main` branch and `/root`
7. Save
8. GitHub will give you a public link for your game

## Notes
I corrected one definition so it is scientifically accurate:
- **Voxel** = the **3D** version of a pixel

If you want to use your original wording instead, edit the text in `script.js`.
